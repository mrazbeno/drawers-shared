# Content
This repo contains shared content, mainly TypeScript types for the WebSocket messaging system of the collaborative drawing app called Drawers. It is used both in [frontend](https://github.com/mrazbeno/drawers-frontend) and the [message router]().

# Usage example
Inside an npm package.json:
```json
{
  "dependencies": {
    "drawers-shared": "github:mrazbeno/drawers-shared#v0.1.0"
  }
}
```
